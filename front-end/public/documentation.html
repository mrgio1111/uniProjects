How To Run The Application

Getting Started

The first thing you need to do is open the terminal and run "docker compose up" to set up your containers (We assume docker is already installed). After running the command the following message will appear:

  • Front-end listening on port 3000
  • Spotify dataset api listening on port 8080
  • Connected to the local spotify database.

Navigating Through The Application

The Back-End

Search "localhost:8080" on your browser. There you will see our swagger specification which you can test yourself. It is easy to navigate and thoroughly explained.

The Front-End

Once more search for "localhost:3000" on your browser. This is a webpage where you can test the functionality of our application. The first thing you will notice is the bolded titles. Each of these titles briefly describe the functionality.

Usage of the Front-End

Look up Track by Name

In this section you will see a blank which should be filled with a name of a track. After filling the name of the track an pressing the button "Retrieve track" you will receive information about that song.

Look up Track by Artist Id

In this section you will see a blank which should be filled with the artistId of the artist who wrote the track. After filling the artistId press the button "Retrieve track" and you will receive information about songs that the artist wrote.

Retrieve Track by ID

In this section you will see a blank which should be filled with the ID of a track. After filling the ID of the track and pressing the button "Retrieve track" you will receive information about that song.

Top N tracks / Top N artists per year

This section will show a list of top n songs or artists in a given year. They are filtered by popularity. First you must select whether you want the top n tracks or artitsts in a box. Afterwards you must select the number of songs you want to see (top n) and the year. Also you can select how many results you want to see on a single page. Lastly click the "Retrieve track" button and the list will be shown.

Create Track by ID

Here you can create a new track by filling the blanks with the appropiate information. There are mostly blanks but there are some other futures s well like checking \ a box or filling a relese date. Fter filling in all the information press on the "Create track" button.

Delete Track by ID

In the blank enter a track ID and click on the "Delete track" button. This will delete the track from our database and it won't be visible anymore.

Delete all tracks by artist ID

To delete all tracks of a specific artist fill in the blank with the artist ID. CLick on the "Delete tracks" button and the operation will be complete.

Retrieve artist summary by artist ID

Enter the artist ID in the blank and press the "Submit" button. This will retrieve the song count, the latest release (song) and its information, the earliest release and its information and the highest popularity.

Update Track

The interface for updating a track shows up when you first request that track. The information of the track is shown and you can change it however you like. The fields are the same as the ones in the Create track.
